# Gocurl
Gocurl will fetch website's index code

## Requirements
* Command Line
* Golang package
* Git installed

## To Use
```shell
$ git clone https://github.com/codingsmartschool/gocurl.git
$ cd gocurl
$ go run setup.go
>> URL:
http://example.com # you have to use http:// because it's a valid url
```
