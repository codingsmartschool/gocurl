# Gocurl
Gocurl will fetch index.html or index.php website's.

# Requirements
* Command Line
* Golang package
* Git installed

## To Use
```shell
$ git clone https://github.com/codingsmartschool/gocurl.git
$ cd gocurl
$ go run setup.go 
>> url: (Make sure to use http://)
http://example.com
```
