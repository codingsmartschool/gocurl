# Gocurl
Gocurl will fetch website's index code

## Requirements
* Command Line
* Golang package
* Git installed

## To Use
```shell
$ git clone https://github.com/codingsmartschool/gocurl.git
$ cd gocurl
$ go run setup.go
>> Insert url: (Make sure to use http://)
http://example.com
```
